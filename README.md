# Dashboard

Interactive web dashboard for monitoring crop health and irrigation conditions in Sonora, Mexico.

## Features

* NDVI & NDWI visualization with seasonal trends
* Field selection (Valle del Yaqui Norte/Sur, Hermosillo, Caborca)
* Simulated IoT sensors (soil moisture, temperature, battery)
* Threshold indicators for crop health classification
* Dark mode support
* Fully responsive design
* No backend required

## How it works

The dashboard uses deterministic simulation based on geographic coordinates to generate realistic agricultural data over a 6-month growing season.

* NDVI → vegetation health
* NDWI → water content
* Sensor panel → real-time farm conditions

## Tech Stack

* HTML, CSS, JavaScript
* Chart.js (via CDN)
